# 👨‍💼 Employee Performance Prediction using Decision Tree

## 📌 Overview
This project predicts whether an employee will achieve performance targets using HR and productivity data.

## 🎯 Objective
- Identify underperforming employees
- Improve productivity planning
- Support HR decision-making

## ⚙️ Features Used
- Experience
- Working Hours
- Project Completion Rate
- Ratings
- Overtime Hours
- Leaves
- Department

## 🧠 Model
- Decision Tree Classifier

## 📊 Evaluation
- Accuracy
- Confusion Matrix
- Classification Report

## 📈 Key Insights
- Some features like ratings and project completion rate strongly influence performance
- Model helps identify employees needing support
